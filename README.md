# Node.js server to upload Images

## To run the project

1.Switch to __node-upload-image__ directory,
```
cd node-upload-image
```
2.Create __images__ directory
```
mkdir images
```
3.Download dependencies
```
npm install
```
4.Run the project
```
node app
```
